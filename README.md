# AutoOsusu

AutoOsusu is an autonomous savings platform inspired by traditional cooperative systems (Osusu), powered by AI agents that manage contributions and payouts without requiring trust between participants.

---

## Overview

AutoOsusu transforms group savings into a trustless, automated system. Each participant is represented by an agent wallet that:

- Contributes funds automatically
- Verifies other participants’ contributions
- Executes payouts based on a rotation system

---

## Agent Autonomy

Each participant is represented by an autonomous agent that independently:

- Checks balance before contributing
- Sends the exact contribution amount
- Verifies other participants’ payments
- Executes payout logic

No manual intervention is required during execution.

---

## Features

- Autonomous agent contributions
- Fixed contribution enforcement
- Rotational payout system
- Real-time transaction logs
- Round duration settings
- Clear recipient tracking per round
